# API Rest AdonisJS 5

This is the default for creating a Rest API in AdonisJs 5, it comes preconfigured with.

1. Controllers
2. Services
3. Repositories
4. Lucid/Database
5. Interface
6. Validators

## Setup

Use the command to install
```bash
yarn start
```

Use the command to execute
```bash
node ace serve --watch
```
